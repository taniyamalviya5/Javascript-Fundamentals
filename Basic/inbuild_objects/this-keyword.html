this keyword
this is a global scope to the global runtime and not to this object. For use the this keyword in the object, we either have to use the standar function function (){} or use without function keyword functionName(){}.
Function Keyword const person = { name: 'Taniya', age: 27, getName: function () { console.log("Hi, I am", this.name); } }
Without function Keyword and arrow const person = { name: 'Taniya', age: 27, getName() { console.log("Hi, I am", this.name); } }
